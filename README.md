---

## 🛠️ Tecnologias Utilizadas

As seguintes tecnologias foram usadas no desenvolvimento do projeto:

- **HTML5** – estrutura das páginas  
- **CSS3** – estilização e layout  
- **JavaScript** – interatividade e funcionalidades  

---
## 🚀 Melhorias Futuras

O projeto ConnectPro poderá evoluir com as seguintes funcionalidades:

### 🔐 Sistema de Autenticação

* Login para profissionais cadastrarem seus serviços
* Login para usuários que desejam contratar serviços
* Cadastro com e-mail e senha
* Recuperação de senha
* Área do usuário com histórico de solicitações

### 👤 Área do Profissional

* Cadastro de serviços oferecidos
* Upload de foto do profissional
* Definição de preço médio
* Definição de área de atendimento
* Edição e exclusão de perfil
* Disponibilidade de horários

### 🔎 Sistema de Busca Avançada

* Busca por categoria (Eletricista, Encanador, etc.)
* Busca por cidade ou região
* Filtro por avaliação
* Filtro por preço
* Filtro por disponibilidade

### ⭐ Sistema de Avaliações

* Usuários avaliarem profissionais
* Sistema de estrelas (1 a 5)
* Comentários dos clientes
* Ranking de melhores profissionais

### 💬 Chat entre Usuário e Profissional

* Chat interno no site
* Solicitação de orçamento
* Envio de mensagens diretas
* Notificações de novas mensagens

### 📅 Agendamento de Serviços

* Calendário de disponibilidade
* Agendamento direto pelo site
* Confirmação automática
* Cancelamento de agendamento

### 📍 Geolocalização

* Mostrar profissionais próximos
* Integração com mapa (Google Maps)
* Cálculo de distância
* Busca por localização atual

### 📱 Melhorias de Interface

* Dark mode (modo escuro)
* Animações mais avançadas
* Melhorias na navegação mobile
* Skeleton loading
* Toast notifications

### 🔔 Sistema de Notificações

* Notificação de novos pedidos
* Notificação de mensagens
* Notificação de avaliação recebida

### 💳 Pagamento Online

* Pagamento dentro da plataforma
* Integração com Pix
* Integração com cartão de crédito
* Comissão da plataforma

### 🛠 Painel Administrativo

* Gerenciamento de usuários
* Gerenciamento de profissionais
* Aprovação de cadastros
* Dashboard com estatísticas
* Controle de avaliações

### 🌎 Outras Melhorias

* Internacionalização (Português / Inglês)
* API própria do sistema
* Integração com WhatsApp
* Favoritar profissionais
* Compartilhar perfil
* Sistema de denúncias
* Upload de portfólio do profissional
